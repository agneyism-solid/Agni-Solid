the World Wide Web Consortium

I’m presently trying to learn XHTML+RDFa, and have included some links below. My initial “how-to” questions were answered by W3C Wiki’s Doctypes and markup styles page, including HTML5 doctype syntax, the main syntax differences between HTML and XHTML, and the .xhtml file extension and XML declaration. Also, Example 2 in Wikipedia’s XHTML article resolves another syntactical question I had about the order of the first two elements.
